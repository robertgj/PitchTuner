template<typename C>
struct SimpleAudio::saSampleFromSource< C >

Normally integral types are shifted to the left of the binary point. For data sources and sinks it is more convenient to think of integral values as lying in the range [-1,1). These are the permitted conversion functors that handle source and sink values.
